Primasindo's Website

This project, named Primasindo's Website, was created as part of my coursework for Software Engineering during my fifth semester in 2023. Website Primasindo serves as an informational platform for a psychology service provider called Primasindo. The website features various elements related to the field of psychology, including a registration feature for counseling sessions and a section dedicated to articles about psychological topics. Additionally, the website is designed to be accessible to a wide audience interested in learning more about Primasindo and its services.

E-Vaksin

This website contains information about COVID-19 vaccination. In 2022, I developed this site to facilitate easy online access for relevant institutions offering vaccination services, aiming to streamline the process for individuals seeking to get vaccinated against COVID-19. One of the key features of this website is the registration system for vaccine recipients. Additionally, it includes a comprehensive information center about COVID-19 vaccination, providing users with valuable insights and updates on vaccination schedules, types of vaccines available, and other related information.

UniBookStore

I created this website to include information on various library resources, one of which is a list of books from different fields accessible to a wide range of people. Among the features I developed for this website is a book search functionality, allowing users to find books by entering the title and author, thus facilitating an easy and efficient search process. Additionally, there is a feature for library administrators to manage the book list, ensuring that it remains updated and consistent for users. This website was developed as part of a project to fulfill the requirements for obtaining a Junior Web Developer certificate from a Professional Certification Institution.

Sistem Informasi Geografis - Yogyakarta

In 2023, I developed the Geographic Information System (GIS) Tourism website as part of my coursework for the Global Positioning System class. The primary aim of this website is to simplify the search for tourist information, especially for both domestic and international travelers visiting Yogyakarta. The website provides information in the form of tourist maps, offering an overview of the layout of various tourist spots in Yogyakarta. Users can easily access this feature from anywhere and at any time. Furthermore, it is designed to provide users with information about the description or history of the tourist destinations they plan to visit, enhancing their overall experience.
